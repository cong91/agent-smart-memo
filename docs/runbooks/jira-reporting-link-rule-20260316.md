# Jira Reporting Link Rule

Date: 2026-03-16

## Rule

Khi báo cáo task/issue/artifact cho MrC:
- không chỉ ghi plain text path hoặc issue key
- phải ưu tiên **bọc link bấm được** để truy cập ngay

## Áp dụng

### Jira issues
- ưu tiên gửi URL issue bấm được
- không chỉ ghi `ASM-93`, `ASM-94`, ... trần

### Artifacts
- nếu có URL/web link thì gửi link bấm được
- nếu chỉ là local path thì ghi rõ path đầy đủ và cách mở nếu cần

## Why

Giảm friction khi kiểm tra tiến độ, review artifact, và mở đúng issue/tài liệu ngay lập tức.
